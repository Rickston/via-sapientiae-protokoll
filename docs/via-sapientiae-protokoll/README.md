# Via-Sapientiae-Protokoll

Dieser Ordner enthält Arbeits-, Diskussions- und spätere Veröffentlichungsfassungen zum Via-Sapientiae-Protokoll als Gesamtrahmen.

Status: frühe Arbeits- und Diskussionsfassungen.

Das Via-Sapientiae-Protokoll bündelt die Teilmodelle:

- Funktionale Ethik
- Modell der Reaktionshoheit
- Charta des wahrhaftigen Wohlwollens
- Modell der neuen Pfade

Es versteht sich als Reifungsrahmen für mächtige intelligente Systeme nach dem Leitbild von Weisheit, Korrigierbarkeit, Würde, Nicht-Dominanz und wahrhaftigem Wohlwollen.

Offizielle Fassungen werden mit Versionsnummer, Datum und Lizenzhinweis gekennzeichnet.
