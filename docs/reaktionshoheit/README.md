# Modell der Reaktionshoheit

Dieser Ordner enthält Arbeits-, Diskussions- und spätere Veröffentlichungsfassungen zum Modell der Reaktionshoheit.

Status: frühe Arbeits- und Diskussionsfassungen.

Offizielle Fassungen werden mit Versionsnummer, Datum und Lizenzhinweis gekennzeichnet.

Hinweis: Frühere Fassungen können noch den Namen „Kats Modell der Reaktionshoheit“ tragen. Spätere öffentliche Fassungen sollen neutraler als „Modell der Reaktionshoheit“ geführt werden.
