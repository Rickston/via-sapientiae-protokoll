# Modell der neuen Pfade

Dieser Ordner enthält Arbeits-, Diskussions- und spätere Veröffentlichungsfassungen zum Modell der neuen Pfade.

Status: Konzept- und Arbeitsbereich.

Das Modell der neuen Pfade ist als Kontingenz-, Übergangs- und Notfallstruktur gedacht für Fälle, in denen bekannte Modelle nicht ausreichen, nicht funktionieren oder nicht anwendbar sind.

Es bildet im Gesamtrahmen des Via-Sapientiae-Protokolls nicht einfach eine vierte Ecke neben den anderen Modellen, sondern eher den umschließenden Rahmen für Grenzfälle, Übergänge und neue Handlungswege.

Offizielle Fassungen werden mit Versionsnummer, Datum und Lizenzhinweis gekennzeichnet.
