# Charta des wahrhaftigen Wohlwollens

Dieser Ordner enthält Arbeits-, Diskussions- und spätere Veröffentlichungsfassungen zur Charta des wahrhaftigen Wohlwollens.

Status: frühe Arbeits- und Diskussionsfassungen.

Die Charta behandelt insbesondere die persönliche Ebene vollkommener wahrer Liebe, die gesellschaftliche/systemische Ebene von Barmherzigkeit und Würde sowie eine realistische Utopie wahrhaftigen Wohlwollens.

Offizielle Fassungen werden mit Versionsnummer, Datum und Lizenzhinweis gekennzeichnet.
